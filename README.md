# AI Reel Script and Thumbnail Generator
# 🎬 AI Creator Assistant

AI-powered short-form content generation platform built using Streamlit, Gemini AI, and Cloudflare AI.

This application helps creators generate:
- Viral video scripts
- Hooks
- Scene structures
- CTAs
- Hashtags
- AI-generated thumbnails

---

# 🚀 Features

## ✨ AI Script Generation
Generate:
- Video Title
- Hook
- Short-form Script
- Scene Structure
- CTA
- Hashtags

using Gemini AI.

---

## 🖼️ AI Thumbnail Generation
Generate 3 cinematic AI thumbnails using Cloudflare AI models.

## 🎨 Modern Purple UI
- Futuristic design
- Responsive layout
- Dark purple AI theme
- Interactive dashboard
